Notes

This is not a perfect Tekken input engine. It is a muscle-memory trainer. It measures your physical input rhythm and consistency.

Keyboard/leverless input is read through keyboard events. Controller/fightstick input is read through the browser Gamepad API. Some devices will only appear if they are in XInput / controller mode.

For KBD with SOCD, this trainer interprets left + right as horizontal neutral. For wave dash, the main expected pattern is now your version: tap f → neutral → d → roll to df → neutral.
